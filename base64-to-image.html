① How to use

1
Paste your Base64 string into the "Base64 Input" field.
2
Press "Decode Base64 to Image" to render the result.
3
Preview the image, check its details, and download it.

ⓘ Important notes

This decoder forces the result to display as an image, even if it's actually a different file type — so if you're not sure your string is an image, the preview will tell you something is wrong.

Works with both raw Base64 and full Data URIs (e.g. data:image/png;base64,...) — the prefix is detected and stripped automatically.
