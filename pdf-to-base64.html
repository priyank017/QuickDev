PDF to Base64

Convert any PDF to a Base64 string — use it as a data URI, embed in HTML, JSON, or XML. Useful when binary files can't be sent directly in text documents. Files up to 50 MB. For the reverse, try Base64 to PDF.

① How to use

1
Choose the source of your PDF file from the "Datatype" field.
2
Paste a URL or select a PDF file from your computer.
3
Select your desired output format from the dropdown.
4
Press "Encode PDF to Base64" to generate the result.
5
Copy or download the result from the output panel.

⇄ Output formats

Plain text
JVBERi0xLjUKJYCBgoMKMSAwIG9i...
Data URI
data:application/pdf;base64,JVBERi0...
HTML Embed
<embed type="application/pdf" src="data:..." />
JSON
{"file":{"mime":"application/pdf","data":"..."}}
